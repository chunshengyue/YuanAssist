# YuanAssist 🕊️

> 如鸢 / 代号鸢 安卓端自动化助手
> 非盈利、无广告，献给所有鸟玩家的自动化工具。

YuanAssist 专为需要完成高频重复操作、以及想要录制和执行攻略表格的玩家设计。应用基于图像识别和无障碍服务，可自动适配不同安卓机型（不支持 iOS 系统）。

[⏬ 下载最新版 APK](https://gitee.com/chunshengyue/yuan-assist-data/releases/download/v1.1.4.3/YuanAssist1.1.4.3.apk)

## 🌟 核心功能

* **📊 表格跟打**：把回合动作整理成表格后交给悬浮窗执行，告别手动凹关。支持文字、图片、辟雍学府 JSON 等多种导入方式。
* **⏺️ 表格录制**：直接操作即可同时记录，录制完成的脚本可以保存、导入和复用，适合生成表格以及洞窟重复操作。
* **👁️ OCR 与图像识别**：一键导入任何形式的攻略表格图片，通过 OCR 转化为可运行脚本；内置模板匹配和颜色匹配，为自动化任务提供保障。
* **🤖 日常自动化**：围绕高频任务打造模块化入口，支持自动刷鸟食、刷 624 等重复操作。
* **🧩 星石拼图**：针对星石背包场景，自动滚动拼图获取完整星石截图，一键统计星石数量，降低整理成本。
* **🌍 攻略社区**：内置社区板块，支持查看并一键导入其他广陵王上传的攻略。现已支持导入MaaYuanShare的作业。支持设备 ID 一键注册，注册后即可成为创作者发布自己的脚本。

## 🚀 快速上手

### 1. 权限设置
YuanAssist 依赖安卓的悬浮窗和无障碍功能运行：
1. 打开应用首页，选择启动战斗版或日常版悬浮窗。
2. 首次运行需授予**悬浮窗权限**。
3. 每次启动运行都需重新授予**无障碍权限**（系统级限制）。
4. 权限授予后，悬浮窗将自动弹出。

### 2. 表格跟打模式
1. 展开战斗版悬浮窗，切换至“跟打模式”。
2. 点击导入，选择图片/文字/已保存的脚本。
3. 可在设置中调整操作延时（建议保守设置稍微长一点）。
4. 点击“开始跟打”，悬浮窗将自动最小化并执行操作。

### 3. 日常任务模式
1. 展开日常版悬浮窗，点击设置，选择需要进行的任务（如：刷鸟食、刷624）。
2. 在游戏中切换到除带有观星入口以外的任意界面，点击开始，助手会自动导航并运行。
3. **星石拼图**：需将星石背包滑动至最上方，然后点击开始。

## ❓ 常见问题 (FAQ)

**Q：如何更新版本？**
**A：** 有新版本时应用内会有公告，点击主页的“检查更新”可选择应用内或浏览器更新。

**Q：跟打时漏掉某个角色的操作怎么办？**
**A：** 可能是延时设置过短，部分机型存在小概率点击无效的情况。可以尝试在设置中增加点击延时。如仍无法解决，请联系作者义诊。

**Q：日常任务（刷鸟食/刷624）中途停止？**
**A：** 1. 网络卡顿导致画面未及时切换。
2. 手机运行卡顿：可在设置中开启低配机型适应，或将延时改为 500ms。
3. 素材不适配：进入主页 `功能测试` -> `上传停止位置截图` -> `选择对应文件` -> `一键替换`，使用自己的截图作为识别素材。

**Q：无障碍模式有风险吗？**
**A：** 存在理论风险，无障碍模式是很敏感的权限，可以操作手机和获取屏幕截图。但 YuanAssist 仅使用无障碍权限模拟针对鸟进行物理点击操作和图像识别，代码已完全开源，可自行查阅。原理相当于连点器，甚至比市面上大部分连点器都要安全（图像识别可以防止误触）。

**Q：能不能在模拟器上运行？**
**A：** 理论上可以，但部分模拟器（如 mumu）可能无法正常获取屏幕截图。模拟器端建议直接使用 [maayuan](https://maayuan.top)。
